Theory
Data Visualization Data visualization is the graphical representation of data using charts, graphs, and plots. It helps in understanding patterns, trends, and relationships in the dataset more effectively than raw data.

Plotting using Pandas Pandas provides built-in plotting functions that are integrated with the Matplotlib library. Using the plot() function, different types of graphs can be generated directly from a DataFrame.

Line Plot A line plot is used to represent data points connected by straight lines. It is useful for showing trends over time or continuous data.

Bar Plot A bar plot represents categorical data using rectangular bars. It is useful for comparing values across different categories.

Histogram A histogram is used to represent the frequency distribution of numerical data. It divides data into bins and shows how many values fall into each bin.

Scatter Plot A scatter plot is used to show the relationship between two variables. Each point represents a pair of values, helping identify correlations or patterns.

Box Plot A box plot is used to display the distribution of data based on quartiles. It helps in identifying median, spread, and outliers in the dataset.

Customization of Plots Plots can be customized by adding titles, labels, legends, and changing colors. This improves readability and makes the visualization more informative.

Conclusion
Thus, different data visualization techniques using Pandas were studied and implemented successfully. Various types of plots such as line, bar, histogram, scatter, and box plots were explored. The experiment demonstrated how visualization helps in better understanding and analysis of data.
